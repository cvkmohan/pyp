Parental Controls
Not all computer activity is appropriate for all users. For example, you might not want your children to have access to certain violent games or to use the computer at certain times of day.
The Parental Controls feature of Windows 7 allows an Administrator user to set limits on the activities of other users
1. There are three types of activities you can limit with Parental Controls: time limits, games, and specific applications.
Setting time limits
1. The Time Limits feature enables you to prevent the account from using the computer on certain days and at certain times.
This can be useful to prevent a child from using the computer when there are no adults available to supervise the activities. You can specify certain days of the week and certain hours within each day
Game Settings
1. The Games setting enables you to specify a rating level, above which all games are blocked. This setting applies not only to already installed games but to new games the user might install.
Not all games have ratings, and you can choose to allow or block games with no rating. You can also block specific games of your choosing, regardless of the rating.
2.The game ratings are determined by the Entertainment Software Ratings Board (ESRB).
