In this video we will see how you can create a new account on Windows 7
The Manage Accounts screen is used to make changes to any user account on the computer.
1. Click the Start button and click Control Panel. The Control Panel window opens.
2. Under the User Accounts and Family Safety heading, click Add or remove user accounts. The Manage Accounts screen appears.
3. Click Create a new account. The Create new Account screen appears.
4. In the New account name box, type the name to be associated with the account.
5. Click the Standard user or Administrator option button.
You may note that the Administrator account has full permission to do anything, including installing and removing software and changing system settings.
Each system must have at least one Administrator account. A Standard account on the other hand, has fewer permissions.
A Standard user cannot install or remove software, and cannot make system changes that affect other users.
For example, a Standard user could adjust his own desktop colors and change his own password, but could not delete someone elseâ€™s user account or install a new version of an application.
6. When you click Create Account, the new account is created.
