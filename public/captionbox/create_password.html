In this video we will see how you can access Your Own User Account Settings and create a password for yourself.
1. Click the Start button and click Control Panel. The Control Panel window opens.
2. Click the User Accounts and Family Safety heading. The User Accounts and Family Safety screen appears.
3. Click User Accounts. The User Accounts screen appears, showing links to common activities you can perform on your own account.
4. Click on Create password. The Create Password screen appears for your account.
5. Type the new password in the New password and Confirm new password boxes.
6. Optionally, type a password hint in the Type a password hint box.
7. Click Create password.
